# Weather Detection App

A simple and interactive Weather Detection Application that provides real-time weather information based on user input or location. This project demonstrates the use of APIs, user interface design, and data handling.

---

## Features

* Search weather by city name
* Displays temperature, humidity, and weather condition
*  Shows real-time weather updates
* User-friendly interface
* Fast and responsive

---

## Technologies Used

* Python 
* API Integration (e.g., OpenWeather API)
* HTML, CSS 
* JSON data handling

---

## Installation

1. Clone the repository:

```
git clone https://github.com/your-username/weather-detection-app.git
```

2. Navigate to the project folder:

```
cd weather-detection-app
```

3. Install dependencies:

```
pip install -r requirements.txt
```

---

## Usage

Run the application:

```
python main.py
```

Then enter a city name to get weather details.
